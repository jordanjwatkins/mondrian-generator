Mondrian Generator

Randomly Generates a Visual Based on the Art of Piet Mondrian

Download this project as a .zip file Download this project as a tar.gz file

I was inspired by day four of Jennifer Dewalt's 180 websites in 180 days to learn more about Piet Mondrian and to see if I could write code to randomly generate pieces that emulate his style.
